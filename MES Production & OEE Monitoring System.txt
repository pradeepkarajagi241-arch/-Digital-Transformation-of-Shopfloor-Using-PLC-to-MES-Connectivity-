Real-Time Manufacturing Execution System with PLC-Based OEE Monitoring
Project Overview

This project is a real-time Manufacturing Execution System (MES) designed to collect production data from PLCs and calculate key manufacturing performance metrics. It provides live monitoring of machine status, production counts, and Overall Equipment Effectiveness (OEE) through a visual dashboard.

The system helps improve shopfloor visibility, identify losses, and support data-driven decision-making in manufacturing environments.

Objectives
Capture real-time production data from PLC signals
Calculate key KPIs: Availability, Performance, Quality, and OEE
Monitor machine productivity and downtime
Visualize production data through dashboards
Improve overall manufacturing efficiency

System Architecture

PLC → Data Acquisition Layer → Database/MES Layer → Dashboard (Power BI / Visualization Tool)

Key Features
Real-time machine data collection from PLC tags
Automatic production count tracking
OEE calculation (Availability × Performance × Quality)
Shift-wise and machine-wise performance analysis
Production loss visibility (downtime, rejects, etc.)

KPIs Calculated
Availability = Operating Time / Planned Production Time
Performance = Actual Output / Theoretical Output
Quality = Good Count / Total Count
OEE = Availability × Performance × Quality

Technologies Used
PLC (Machine Data Source)
Industrial Communication (OPC / Ethernet / Tag-based communication)
Python / SQL (Data processing – if applicable)
Power BI / Dashboard Tool (Visualization)

Results
Real-time production monitoring achieved
Accurate OEE tracking per machine/shift
Improved visibility of production losses
Faster decision-making for shopfloor operations

This project was created to demonstrate MES and Digital Transformation (DX) concepts used in smart manufacturing environments.
